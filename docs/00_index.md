# Documentacion Raya Lucaria

Bienvenido a la documentacion del framework **Raya Lucaria**, el generador de sitios estaticos para materiales de cursos del ITAM.

Esta documentacion se organiza en tres secciones:

- **Guia del Estudiante** -- Como navegar y usar el sitio del curso
- **Guia del Profesor** -- Como crear y publicar contenido
- **Agent Guide** -- Technical reference for AI agents working with the codebase
