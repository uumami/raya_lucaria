# Accesibilidad

## Tamano de texto

Puedes ajustar el tamano del texto usando el boton de tamano de fuente en la barra superior:

- **Normal** -- Tamano predeterminado
- **Grande** -- Texto mas grande para mejor legibilidad
- **Extra grande** -- Tamano maximo

## Fuente OpenDyslexic

Si tienes dislexia o prefieres una fuente mas legible, puedes activar la fuente **OpenDyslexic** desde el boton de fuente en la barra superior. Esta fuente fue disenada especificamente para mejorar la legibilidad en personas con dislexia.

## Navegacion por teclado

El sitio es completamente navegable con teclado:

- **Tab** -- Mover entre elementos interactivos
- **Enter** -- Activar enlaces y botones
- **Flechas izquierda/derecha** -- Pagina anterior/siguiente
- **Escape** -- Cerrar menus o dialogos abiertos

## Preferencias guardadas

Todas las configuraciones de accesibilidad (tamano de texto, fuente, tema) se guardan automaticamente en tu navegador y persisten entre sesiones.
