# Proyecto Final

:::project{id="proyecto-final" title="Sitio de Curso" due="2026-05-20" points="100"}

## Descripcion

Crea un sitio web completo para un curso usando el framework Raya Lucaria.

## Requisitos

1. Al menos 5 paginas de contenido
2. Uso de al menos 3 tipos de componentes
3. Incluir matematicas (KaTeX)
4. Incluir al menos un diagrama (Mermaid)
5. Tema personalizado

## Entrega

- Repositorio en GitHub
- Sitio desplegado en GitHub Pages
- Documento de reflexion (1 pagina)

## Evaluacion

| Criterio | Puntos |
|----------|--------|
| Contenido | 30 |
| Componentes | 20 |
| Matematicas y diagramas | 20 |
| Tema personalizado | 15 |
| Documentacion | 15 |
