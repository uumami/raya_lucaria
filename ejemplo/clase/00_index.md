# Bienvenido al Curso de Ejemplo

Este es un curso de ejemplo para demostrar las capacidades del framework **Raya Lucaria**.

## Contenido

Navega usando la barra lateral para explorar los temas del curso.

## Caracteristicas

- Componentes (tareas, ejercicios, prompts, ejemplos, examenes, proyectos)
- Matematicas con KaTeX
- Diagramas con Mermaid
- Busqueda con Pagefind
- Temas oscuro y claro
- Navegacion con teclado
