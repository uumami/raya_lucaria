# Bienvenida al Curso

Bienvenido al curso de ejemplo. Aqui aprenderemos a usar el framework Raya Lucaria.

![Logo de Raya Lucaria](./images/test-image.png)

> **Nota:** Este curso usa el framework *Raya Lucaria*, inspirado en la Academia de Raya Lucaria de Elden Ring. Todo el contenido se escribe en Markdown y se transforma automaticamente en un sitio web.

## Objetivos

1. Entender la estructura del curso
2. Completar la primera tarea
3. Practicar con ejercicios
4. Explorar las [matematicas](../02_avanzado/03_matematicas/) y los [diagramas](../02_avanzado/04_diagramas/)

## Primera Tarea

:::homework{id="tarea-01" title="Configuracion Inicial" due="2026-03-01" points="10"}

Configura tu entorno de desarrollo:

1. Instala Docker
2. Clona el repositorio
3. Ejecuta `docker compose up dev`
4. Verifica que el sitio funcione en `http://localhost:3000`

**Entrega:** Captura de pantalla mostrando el sitio funcionando.

:::

## Ejercicio de Practica

:::exercise{title="Hola Mundo" difficulty="1"}

Crea un archivo `hola.md` en la carpeta `clase/` con el siguiente contenido:

```markdown
---
title: "Mi Primera Pagina"
---

# Hola Mundo

Esta es mi primera pagina.
```

Verifica que aparezca en la navegacion.

:::
